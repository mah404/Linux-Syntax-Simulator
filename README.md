# Linux Syntax Simulator

A beginner-friendly **Linux syntax simulator** and **Linux command line practice tool** built with HTML, CSS, and JavaScript.

Learn Linux commands in the browser by practicing common terminal syntax like `pwd`, `ls`, `cd`, `mkdir`, `touch`, and `cat` in a safe fake shell environment. This project is useful for anyone searching for:

- Linux syntax practice
- Linux command simulator
- Linux terminal simulator
- Linux commands for beginners
- Bash command practice
- Learn Linux command line online

## Features

- Interactive browser-based Linux terminal simulator
- Beginner Linux lessons with guided tasks
- Safe practice environment with no real file changes
- Helpful hints for each exercise
- Simple interface for learning Linux command syntax

## Linux commands covered

- `pwd` — show the current directory
- `ls` and `ls -l` — list files
- `cd` — move between folders
- `mkdir` — create directories
- `touch` — create files
- `cat` — read file contents

## How to use the Linux syntax simulator

1. Open `index.html` in your web browser.
2. Pick a lesson from the left side.
3. Type commands into the terminal area.
4. Use the **Show hint** button if you get stuck.
5. Use **Reset simulator** to start over.

## Project files

## Why this project exists

Learning Linux syntax can feel difficult at first because many commands are short, symbolic, and easy to forget. This simulator gives beginners a simple place to practice Linux terminal commands before using a real shell.

## Notes

- This is a learning simulator, not a real shell.
- Commands only affect the fake practice environment inside the page.
- No real files or folders on your computer are changed.
